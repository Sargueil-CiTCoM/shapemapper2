Parse a single BAM/SAM alignment.

Parameters
left_target_posLeftmost alignment target position covered by a read (0-based target coord), typically from BamTools::BamAlignment.Position
right_target_posRightmost alignment target position covered, typically from BamTools::BamAlignment.GetEndPosition()-1
query_basesRead sequence, typically from BamTools::BamAlignment.QueryBases
cigar_dataParsed CIGAR string, typically from BamTools::BamAlignment.cigar_data
md_tag_contentsMD tag, typically from BamTools::BamAlignment.GetTag("MD", ...)
Returns
Returns vector containing: Left-most position in alignment target sequence (0-based), right-most position (0-based), reconstructed target sequence over alignment region, and mutations with respect to the target sequence. Mutations are also adjusted so that ambiguously aligned indels include information about alternate valid placements (see identifyAmbiguousMutations()). Now using Read class.
